## 🎓 HOW TO USE

### Add New Product (5 minutes)
1. Open `products-data.js`
2. Find the category array (e.g., `'home-appliances'`)
3. Add new product object (copy existing as template)
4. Fill in all required fields
5. Save and refresh browser

### Update Price (2 minutes)
1. Open `products-data.js`
2. Find product by name
3. Update `price` and `originalPrice`
4. Discount calculates automatically
5. Save and refresh

### Change Product Image (3 minutes)
1. Open `products-data.js`
2. Find product
3. Replace `image` URL
4. Use 300x300px images
5. Save and refresh

### Customize Colors (5 minutes)
1. Open `styles.css`
2. Find `:root { }` CSS variables
3. Update hex color codes
4. Test changes in browser
5. Save

### Add New Category (10 minutes)
1. Open `products-data.js`
2. Add new category key
3. Add products array
4. Update HTML if needed
5. Save and refresh
