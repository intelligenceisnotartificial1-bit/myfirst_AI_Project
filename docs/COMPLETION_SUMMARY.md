```

2. Scroll to Categories

4. Click any product
5. Add to cart
6. View cart
7. Try checkout
```

### Add New Products
```javascript
// Edit products-data.js
'home-appliances': [
  // ... existing products
  {
    id: 'HA007',
    name: 'New Product',
    price: 29999,
    // ... other fields
  }
]
```

### Update Prices
```javascript
// Edit products-data.js
price: 29999,           // Current price
originalPrice: 39999,   // Original (auto-calculates discount)
```

### Change Colors
```css
/* Edit styles.css */
:root {
  --primary: #6366f1;      /* Your color */
  --secondary: #8b5cf6;    /* Your color */
}
```
