## 🔧 Customization Guide

### Add Products
**File:** `products-data.js`
```javascript
{
    id: 'HA007',
    name: 'Product Name',
    category: 'home-appliances',
    price: 29999,
    originalPrice: 39999,
    rating: 4.8,
    reviews: 1000,
    image: 'https://image-url.jpg',
    description: 'Description',
    stock: 50,
    badge: 'New',
    features: ['Feature 1', 'Feature 2'],
    demand: 'Very High'
}
```

### Update Prices
**File:** `products-data.js`
```javascript
price: 29999,           // Current price
originalPrice: 39999,   // Original price (discount auto-calculated)
```

### Change Images
**File:** `products-data.js`
```javascript
image: 'https://your-image-url.jpg'
```

### Update Colors
**File:** `styles.css`
```css
:root {
    --primary: #6366f1;      /* Main color */
    --secondary: #8b5cf6;    /* Secondary color */
    --accent: #ec4899;       /* Accent color */
}
```
